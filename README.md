#  Gestor de Alumnos en Java

Mi primer proyecto básico de consola hecho en Java para gestionar alumnos, realizado durante mis estudios de DAM.


## Funcionalidades

- Añadir alumnos con nombre y nota.
- Mostrar listado de alumnos.
- Buscar alumnos por nombre o nota.
- Eliminar alumnos por nombre o nota.
- Calcular media de notas.
- Contar aprobados y suspendidos.

##  Tecnologías usadas

- Java
- ArrayList
- Scanner
- Iterator
- Programación estructurada
